The file **`stops.csv`** contains a list of all active stops for the Corfu City Bus (Astiko KTEL Kerkyras), including their IDs, their geographical coordinates, and the stop names in both Greek and English.

### `stops.csv` Structure

| Column | Description |
| :--- | :--- |
| **ID** | The unique numerical stop code used by the Corfu City Bus system. |
| **EN\_NAME** | The English name of the stop.. |
| **GR\_NAME** | The Greek name of the stop. |
| **LAT** | The Latitude coordinate of the stop. |
| **LONG** | The Longitude coordinate of the stop. |
