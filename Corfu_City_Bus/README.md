# 🚌 Corfu City Bus - Αστικό ΚΤΕΛ Κέρκυρας

The file **`stops.csv`** contains a list of all active stops for the Corfu City Bus (Astiko KTEL Kerkyras), including their IDs, their geographical coordinates, and the stop names in both Greek and English.

### `stops.csv` Structure

| Column | Description |
| :--- | :--- |
| **ID** | The unique numerical stop code used by the Corfu City Bus system. |
| **EN\_NAME** | The English name of the stop.. |
| **GR\_NAME** | The Greek name of the stop. |
| **LAT** | The Latitude coordinate of the stop. |
| **LONG** | The Longitude coordinate of the stop. |

### 🔗 Useful Links

* **Official Corfu City Bus Website (Astiko KTEL Kerkyras):** [https://astikoktelkerkyras.gr/en](https://astikoktelkerkyras.gr/en)
* **All Stops:** To find and view **all stops** of the Corfu Urban Bus (Astiko KTEL) on the map, visit: [https://corfu.citybus.gr/en/stops/](https://corfu.citybus.gr/en/stops/)
* **Live Departures (Example):** To view **real-time** arrivals/departures for a specific stop (e.g., ID `201`), visit: [https://corfu.citybus.gr/en/stops/live/201](https://corfu.citybus.gr/en/stops/live/201)
